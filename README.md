# [Live Site](https://parts-bd.web.app/)

# Project Features

## User Context

-   Login using Email/Google
-   My Profile
-   Update Profile
-   Place Order
-   Order Review
-   Order Cancel
-   Payment using Card
-   Getting Status of Order (pending, shipped etc)
-   Adding a review

## Admin Context

-   My Profile
-   User management
-   Order Management
-   Product Management
-   Add Product

# Tools & Technology Used

-   Firebase Auth
-   Axios (+ interceptor)
-   React query
-   React router dom
-   React countdown
-   React rating star
-   TailwindCSS
-   DaisyUI
-   Require Auth both Admin and User level
-   Git Version Control + GitHub
-   Heroku (Server)
-   Firebase Hosting (Client)
